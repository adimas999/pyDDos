### Install
Clone this project

```bash
> git clone https://github.com/adimas999/pyDDos
> cd pyDDos
```

Install the dependencies:

```bash
> pkg update
> pkg upgrade
> pkg install python -y
> pkg install python2
```

### Usage
run the DDOS Script

```bash
> python DDOS.py
Or
> python2 DDOS.py
```
